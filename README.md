# 庄子 技能包

两千三百年前的生存手册，蒸馏为可执行的 Claude Code 技能。

## 安装

```
/install gh:Nelthari0n/worldcraft
```

## 使用

```
/zhuangzi              # 描述处境，自动匹配策略
/zhuangzi 处世         # 人间世：危险环境中的生存
/zhuangzi 破局         # 逍遥游：突破自我设限
/zhuangzi 认知         # 齐物论：超越二元对立
/zhuangzi 精进         # 养生主：顺势精准行动
/zhuangzi 影响         # 德充符：不靠权力影响人
/zhuangzi 应变         # 大宗师：面对不可控巨变
/zhuangzi 领导         # 应帝王：无为而治
```

## 七大策略体系

| 策略 | 来源 | 核心问题 | 一句话 |
|------|------|----------|--------|
| **处世术** | 人间世 | 如何在危险复杂的人际环境中生存 | 外化而内不化 |
| **破局术** | 逍遥游 | 如何识别并突破自我设限 | 你以为的边界只是框架 |
| **认知术** | 齐物论 | 如何跳出非此即彼的陷阱 | 不是谁对，是用哪个框架 |
| **精进术** | 养生主 | 如何做到顺势精准的行动 | 以无厚入有间 |
| **影响术** | 德充符 | 如何不靠权力和外在影响他人 | 内在稳定就是最强吸引力 |
| **应变术** | 大宗师 | 如何面对不可逆的巨大变化 | 安时处顺，在新现实中找新可能 |
| **领导术** | 应帝王 | 如何管理而不控制 | 最好的领导让系统自运转 |

### 处世术包含五个子策略

| 子策略 | 原典 | 一句话 |
|--------|------|--------|
| 心斋术 | 颜回见孔子 | 想改变别人，先清空自己 |
| 不可能任务 | 叶公子高使齐 | 做不到的事，做到你能做的部分 |
| 伴虎术 | 颜阖为太子师 | 外在顺从，内心独立 |
| 散木策略 | 栎社树 | 不被看见，才能活得久 |
| 以缺为盾 | 支离疏 | 有策略的不完美是保护伞 |

## 设计原则

- **可执行，非说教** — 每个策略都有情境触发、行动步骤、现代示例、反模式警告
- **互相联通** — 每个策略标注退出条件和切换建议，形成网络而非孤岛
- **不美化庄子** — 这些是生存策略，不是心灵鸡汤。有代价，有边界，有不适用的时候
- **原典为锚** — 每个技能以庄子原文寓言为锚点，不是凭空发挥
