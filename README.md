# 庄子·人间世 技能包

在危险复杂的人际环境中生存与行事的五种策略，提炼自《庄子·内篇·人间世》。

## 安装

```
/install /path/to/renjiashi-skills
```

## 使用

```
/renjiashi           # 描述你的处境，自动匹配策略
/renjiashi 心斋      # 劝说强势对象前的自我清空
/renjiashi 使命      # 进退两难时的承接之道
/renjiashi 伴虎      # 与危险人物共事的生存法则
/renjiashi 散木      # 以"无用"求生存
/renjiashi 支离      # 用不完美保护自己
```

## 五种策略

| 策略 | 原典 | 一句话 |
|------|------|--------|
| 心斋术 | 颜回见孔子 | 想改变别人，先清空自己 |
| 不可能任务 | 叶公子高使齐 | 做不到的事，做到你能做的部分 |
| 伴虎术 | 颜阖为太子师 | 外在顺从，内心独立 |
| 散木策略 | 栎社树 | 不被看见，才能活得久 |
| 以缺为盾 | 支离疏 | 有策略的不完美是保护伞 |

## 设计原则

- **可执行，非说教**：每个策略都有明确的情境触发、行动步骤、反模式警告
- **互相联通**：每个策略都标注了退出条件和切换建议
- **不美化庄子**：这些是生存策略，不是心灵鸡汤。有代价，有边界，有不适用的时候
