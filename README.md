# Worldcraft — 智慧蒸馏技能包

把经典智慧蒸馏为可执行的 Claude Code 技能。不是读书笔记，是决策工具。

## 安装

```
/install gh:Nelthari0n/worldcraft
```

---

## 庄子技能包 `/zhuangzi`

两千三百年前的生存手册。

```
/zhuangzi              # 描述处境，自动匹配策略
/zhuangzi 处世         # 人间世：危险环境中的生存
/zhuangzi 破局         # 逍遥游：突破自我设限
/zhuangzi 认知         # 齐物论：超越二元对立
/zhuangzi 精进         # 养生主：顺势精准行动
/zhuangzi 影响         # 德充符：不靠权力影响人
/zhuangzi 应变         # 大宗师：面对不可控巨变
/zhuangzi 领导         # 应帝王：无为而治
```

| 策略 | 来源 | 一句话 |
|------|------|--------|
| **处世术** | 人间世 | 外化而内不化 |
| **破局术** | 逍遥游 | 你以为的边界只是框架 |
| **认知术** | 齐物论 | 不是谁对，是用哪个框架 |
| **精进术** | 养生主 | 以无厚入有间 |
| **影响术** | 德充符 | 内在稳定就是最强吸引力 |
| **应变术** | 大宗师 | 安时处顺，在新现实中找新可能 |
| **领导术** | 应帝王 | 最好的领导让系统自运转 |

处世术下含五个子策略：心斋术、不可能任务、伴虎术、散木策略、以缺为盾。

## 芒格技能包 `/munger`

查理·芒格的思维与决策工具箱。

```
/munger                # 描述问题，自动匹配工具
/munger 反演           # 反过来想，总是反过来想
/munger 误判           # 25种认知偏差识别与反制
/munger 模型           # 跨学科多元思维模型格栅
/munger 能力圈         # 知道自己不知道什么
/munger 清单           # 重大决策前的系统化检查
/munger lollapalooza   # 多因素共振的极端效应
```

| 工具 | 核心 | 一句话 |
|------|------|--------|
| **反演术** | 研究怎么失败 | 如果我知道我会死在哪里，就永远不去那里 |
| **误判心理学** | 25 种认知偏差 | 你的大脑在骗你，这是清单 |
| **多元模型** | 跨学科分析格栅 | 拿着锤子看什么都像钉子 |
| **能力圈** | 知识边界判断 | 熟悉不等于理解 |
| **决策清单** | 系统化检查流程 | 飞行员不会因为经验丰富就不看清单 |
| **Lollapalooza** | 多因素共振 | 极端结果来自多种力量同时叠加 |

## 两个体系的关系

| 芒格 | 庄子 | 共通点 |
|------|------|--------|
| 反演 | 齐物论 | 视角翻转 |
| 能力圈 | 濠梁之辩 | 认知谦逊 |
| 避免愚蠢 > 追求聪明 | 无用之用 | 减法思维 |
| 多元模型 | 庖丁解牛 | 穿透表面的认知 |

用芒格做理性分析，用庄子处理理性之外的不确定性。

---

## 设计原则

- **可执行，非说教** — 每个策略都有情境触发、行动步骤、现代示例、反模式警告
- **互相联通** — 退出条件指向其他更合适的策略，形成网络
- **有边界** — 每个策略都标注不适用的场景和切换信号
- **原典为锚** — 庄子以寓言为锚，芒格以原话和案例为锚
