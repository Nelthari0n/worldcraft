# 能力圈 — 知道自己不知道什么

> "知道自己能力的边界比能力本身更重要。在你的能力圈内部，你几乎不可能失败。在你的能力圈外部，你几乎不可能成功。"

## 核心理念

芒格和巴菲特几十年来拒绝了无数"好机会"。不是因为那些机会不好，是因为他们判断自己不够懂。

**能力圈不是说你知道多少，是说你知道你不知道什么。**

三种人：
1. 知道自己知道什么——能做对的事
2. 知道自己不知道什么——能避免做错的事
3. 不知道自己不知道什么——最危险的状态

芒格的名言："我们遇到的大部分灾难，都发生在能力圈之外。人们以为自己懂了，其实只是熟悉了。**熟悉 ≠ 理解。**"

## 情境触发

当你遇到以下场景时，使用此技能：

- 被诱惑去做一个你"大概懂"但没有深度经验的事
- 别人说机会很好，你心动了但不确定
- 你要在一个领域做重大投入（时间、金钱、声誉）
- 有人问你意见，你想给出一个聪明的回答但其实不太确定
- 你在评估一个团队/合伙人——他们是否在自己的能力圈内

## 行动步骤

### 第一步：画你的圈——诚实地

**真正的能力圈有三层：**

```
┌──────────────────────────────────┐
│      不知道自己不知道的           │ ← 盲区（最危险）
│  ┌───────────────────────────┐   │
│  │    知道自己不知道的        │   │ ← 安全区（知道该问谁）
│  │  ┌─────────────────────┐  │   │
│  │  │  真正知道的          │  │   │ ← 能力圈（可以下注）
│  │  └─────────────────────┘  │   │
│  └───────────────────────────┘   │
└──────────────────────────────────┘
```

**判断你是否"真正知道"的测试：**
1. 你能用通俗语言解释给外行听吗？（如果不能，你可能只是会用术语）
2. 你能预测这个领域里什么会失败吗？（知道失败模式=真的懂）
3. 你经历过这个领域的完整周期吗？（只见过牛市不算懂投资）
4. 如果这件事出问题了你知道怎么应对吗？（只知道顺利时怎么做不够）
5. 你能识别这个领域里的"伪专家"吗？（如果不能，你自己可能就是）

### 第二步：区分"熟悉"和"理解"

**这是最关键的一步。** 芒格说大多数灾难来自把"熟悉"误认为"理解"。

| 熟悉 | 理解 |
|------|------|
| 我看过很多文章 | 我自己做过并且知道坑在哪里 |
| 我能说出关键概念 | 我能用这些概念解决实际问题 |
| 我认识这个领域的人 | 我自己有独立判断力 |
| 我知道过去发生了什么 | 我知道什么条件下会产生不同的结果 |
| 我有成功的经历 | 我知道为什么成功以及什么条件下会失败 |

**操作：**
对你即将投入的领域，逐条对照上表。如果大部分在左栏——你只是熟悉，不是真的懂。

### 第三步：圈外行动的规则

不是说能力圈外的事就不能做。但规则不同：

**圈内（你真正懂的领域）：**
- 可以重仓——大量投入时间、金钱、声誉
- 可以快速决策——你有足够的判断力
- 可以相信自己的直觉——直觉在深度经验领域是可靠的

**圈外（你不够懂的领域）：**
- 小仓位——只投入你能承受全部损失的量
- 慢决策——给自己更多时间收集信息
- 不信直觉——你的直觉在不熟悉的领域会骗你
- 找向导——找到这个领域真正在圈内的人，听他们的判断
- 安全边际加倍——因为你对风险的判断不如圈内人准确

### 第四步：扩大能力圈（但不要假装已经扩大了）

能力圈可以扩大，但有条件：

**扩大的正确方式：**
1. 选择和你现有能力圈相邻的领域（不要跳远）
2. 投入大量时间学习——读、做、犯错、复盘
3. 找到这个领域的"负面知识"——什么会失败、什么是陷阱
4. 经历至少一个完整周期
5. 达到"能识别伪专家"的程度

**扩大的错误方式：**
- 看了几本书就觉得懂了
- 做了一次就觉得有经验了（幸存者偏差）
- 别人说你懂你就信了（社会认同偏差）

**芒格扩大能力圈的速度：几十年扩大了有限的几个领域。** 不要指望几个月就能扩大。

### 第五步：用"圈外"换诚实

在能力圈之外最有力的话是：**"这个我不懂。"**

**操作：**
- 被问到不懂的问题时——"这个超出我的判断范围，我不好说"
- 面对超出能力的机会时——"这看起来很好，但我不够懂，不敢做"
- 发现自己在能力圈外时——立即降低确定性，加大安全边际

**芒格说：** "巴菲特和我不是通过做正确的很多事来成功的。我们是通过很少犯大错来成功的。而很少犯大错的关键就是待在能力圈内。"

## 示例

**场景**：朋友拉你投资一个 AI 创业项目。你是程序员，懂技术，觉得自己"应该能看懂"。

**能力圈分析：**
1. 画圈：你懂软件工程（圈内），对 AI/ML 读过很多文章（熟悉），对创业投资没有经验（圈外）
2. 熟悉 vs 理解：你能用 PyTorch 训练一个模型吗？你能判断这个 AI 团队的技术路线是否可行吗？你知道 AI 创业常见的死法吗？——如果答案大多是"不太确定"，你只是熟悉，不是理解
3. 圈外规则：只投你能承受全部损失的金额。不要因为"我是程序员所以我能看懂 AI"就加大投入。找一个真正做 AI 的人和一个有投资经验的人各自独立给你意见
4. 扩圈判断：如果你真的想进入这个领域，先花 6-12 个月深入学习和实操，再做大额投入。不要边投边学
5. 诚实："这个项目可能很好，但我对 AI 创业的理解不够深，所以我只放一个小金额，当学费"

## 反模式

| 错误 | 为什么错 |
|------|----------|
| "我很聪明所以学什么都快" | 聪明和在特定领域有深度判断力是两回事 |
| "我认识这个领域的专家" | 你认识专家 ≠ 你有专家的判断力。而且你能判断他是真专家吗？ |
| "上次成功了所以我懂" | 一次成功可能是运气。你知道为什么成功吗？知道什么条件下会失败吗？ |
| "能力圈太小会错过机会" | 芒格只投了几十家公司，错过了无数"好机会"——但没有犯大错 |
| 用"能力圈"当不作为的借口 | 能力圈内的事你必须果断行动。不是所有事都在圈外 |

## 退出条件

- 你确实在能力圈内但不自信 → 这不是能力圈问题是自信问题
- 你必须在能力圈外行动（没有选择）→ 遵守圈外规则：小仓位、慢决策、找向导、加倍安全边际
- 你在用能力圈合理化"我什么都不做" → 芒格不是消极的。圈内的事他非常果断
