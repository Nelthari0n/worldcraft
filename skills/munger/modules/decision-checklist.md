# 决策清单 — 重大决策前的系统化检查

> "飞行员不会因为经验丰富就不看检查清单。清单的价值不是教你新东西，是防止你在压力下遗漏已知的东西。"

## 核心理念

芒格是"清单主义者"。他认为人的大脑天生不可靠——在压力、兴奋、疲劳、社交影响下，即使是最聪明的人也会遗漏显而易见的事。

清单不是给新手的拐杖，是给专家的安全网。

**用法：在做任何不可逆的重大决策之前，过一遍这个清单。不要凭记忆，一条一条核对。**

## 情境触发

当你即将做以下类型的决策时，使用此清单：

- 投资决策（金钱、时间、或声誉的重大投入）
- 战略选择（方向性的、影响未来 1 年以上的）
- 不可逆决定（做了就很难回头的）
- 高风险决定（失败的代价很大的）
- 感觉"显然应该做"的决定（越"显然"越需要检查）

## 决策清单

### A. 前置检查——我应该做这个决定吗？

- [ ] **我现在的状态适合做重大决定吗？**
  - 不在疲劳、醉酒、极度兴奋或极度恐惧的状态下做不可逆决定
  - 如果不适合——推迟。"没有决定"通常比"坏决定"好

- [ ] **这件事在我的能力圈内吗？**
  - 我对这个领域的理解达到"能预测失败模式"的程度了吗？
  - 如果不在圈内→ 应用能力圈的"圈外规则"

- [ ] **有紧迫性吗？必须现在决定吗？**
  - 如果不紧迫——等一等。时间是清晰思考的朋友
  - 如果有人在催你快决定——这本身是一个警告信号

### B. 反演检查——这件事会怎么失败？

- [ ] **我列出了至少 5 种失败模式吗？**
  - "这件事最可能怎么搞砸？"

- [ ] **我做了 Pre-Mortem 吗？**
  - "假设 1 年后这件事已经失败了，最可能的原因是什么？"

- [ ] **我研究过类似的失败案例吗？**
  - "做过类似的事的人，最常见的失败原因是什么？"

- [ ] **如果关键假设错了 50%，结果还能接受吗？**
  - 安全边际检查——不要假设一切都按最好的情况发展

### C. 偏差检查——我的判断被什么扭曲了？

- [ ] **我在这件事里有利益关系吗？**
  - 激励偏见——利益会让你不自觉地美化判断

- [ ] **我的判断受到周围人影响了吗？**
  - 社会认同——"大家都觉得应该做"不等于应该做

- [ ] **我有沉没成本吗？**
  - "如果从零开始，面对当前的信息，我还会做同样的选择吗？"

- [ ] **我是否对正面信息过度兴奋了？**
  - 如果这件事看起来"太好了"——什么被遮蔽了？

- [ ] **我是否在逃避什么而不是追求什么？**
  - 恐惧驱动的决策和欲望驱动的决策都需要额外审视

### D. 模型检查——我用了足够多的视角吗？

- [ ] **我至少从 3 个不同学科的角度看过这个问题吗？**
  - 激励机制怎么说？复利效应怎么说？进化论怎么说？

- [ ] **有没有一个我完全没考虑过的视角？**
  - 最危险的盲点是你不知道你没看到的

- [ ] **我问过持反对意见的聪明人吗？**
  - 不是随便找人反对——找你信任的、在这个领域懂的、且会说真话的人

- [ ] **红队测试：如果有人要反驳我，最强的论据是什么？**
  - 自己先把最强反对意见想出来

### E. 执行检查——如果做了，然后呢？

- [ ] **最坏情况我能承受吗？**
  - 不要赌你承受不起的东西。永远不要

- [ ] **退出机制是什么？**
  - 在什么条件下我会停止/撤退？提前定好，不要到时候再决定

- [ ] **二阶效应是什么？**
  - 做了这件事之后会引发什么连锁反应？好的和坏的

- [ ] **机会成本是什么？**
  - 做这件事意味着不能做什么？那些"不能做的"值多少？

- [ ] **我能清楚地向一个聪明的外行人解释为什么要做这件事吗？**
  - 如果解释不清楚——可能是你自己也没想清楚

### F. 最终检查

- [ ] **过了这个清单之后，我仍然想做吗？**
  - 是→ 做，并且要果断
  - 否→ 不做，不要后悔
  - 不确定→ 再等等。强制给自己 48 小时冷静期

## 使用方式

### 快速模式（15 分钟）

适用于中等重要的决策：
1. 过 A 区（前置）+ C 区（偏差）= 5 分钟
2. 过 B 区（反演）重点看 Pre-Mortem = 5 分钟
3. 过 E 区（执行）重点看最坏情况和退出机制 = 5 分钟

### 完整模式（1-2 小时）

适用于重大不可逆决策：
1. 全部 6 区逐条检查
2. 找至少一个人独立做 D 区的反对意见测试
3. 做完后给自己 48 小时冷静期再最终确认

## 示例

**场景**：你收到一个 offer，薪资比现在高 40%，但要去一个你不太熟悉的行业。

**过清单：**
- A1 状态：现在是冷静的✓
- A2 能力圈：新行业在圈外，需要按圈外规则评估
- A3 紧迫性：有时间限制但不是24小时内必须决定✓ 争取几天
- B1 失败模式：新行业不适应、管理风格水土不服、承诺的项目取消、文化冲突
- B4 假设错 50%：如果薪资涨幅不如预期（期权缩水/奖金波动），还想去吗？
- C1 利益：40% 涨薪在激励我美化这个机会——注意
- C3 沉没成本：在现公司5年——"如果我刚毕业，面对这两个选择，我选哪个？"
- D3 反对意见：找在目标行业工作过的人问"去那边最可能踩的坑是什么？"
- E1 最坏情况：进去半年发现不适合→ 能再找工作吗？经济上能撑多久？
- E2 退出机制：如果 1 年内不达到什么标准，怎么办？
- E4 机会成本：留在现公司继续发展会怎样？

## 反模式

| 错误 | 为什么错 |
|------|----------|
| 用清单证明自己想做的事是对的 | 清单是发现问题的，不是确认已有结论的。如果你发现自己在合理化每条检查项——这就是一个信号 |
| 过了清单但不执行发现的问题 | 发现风险后不做预防=没过清单 |
| 对所有决策都过完整清单 | 午饭吃什么不需要。只用于重大不可逆决策 |
| 清单过了但直觉强烈反对 | 不要忽视直觉——它可能在告诉你清单没覆盖的东西。但要区分"有信息的直觉"和"恐惧" |

## 退出条件

- 决策很小且可逆 → 不需要清单，直接做，错了再改
- 你在用清单拖延决策 → 设截止时间。到了就做决定
- 清单结果清晰但你仍然犹豫 → 问题不在分析，可能在勇气或价值观层面
