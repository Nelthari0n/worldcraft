# 影响术 — 不靠外在条件影响他人

> 《德充符》+《说剑》——真正的影响力不来自你拥有什么，来自你是什么。

## 原典

### 王骀——断足而服人心

鲁国有个叫王骀的人，被砍去了一只脚。但跟他学习的人和孔子的弟子一样多。

"立不教，坐不议"——他不主动教导，不发表议论。但人们"虚而往，实而归"——空着来，满着走。

孔子的弟子问：他残缺了身体，既不说教也不讨论，为什么那么多人追随他？

答案："死生亦大矣，而不得与之变。虽天地覆坠，亦将不与之遗。审乎无假而不与物迁。"——连生死这么大的事都动摇不了他的内心，他还会被什么外在的东西改变？

**核心洞察：当一个人不再被外在条件定义和动摇，反而产生了最强的吸引力。**

### 哀骀它——丑到极致反而让人无法离开

哀骀它是一个极其丑陋的人。但男人和他相处后不想离开，女人见了他对父母说"宁为夫子妾"。他没有权力给人好处，没有财富，长得丑，从不主动说话——为什么？

庄子说："才全而德不形。"——内在完整但不外显。他不试图展示什么、证明什么、改变什么。正是这种"不求"，让所有人信任他。

**核心洞察：影响力最大的人不是表现最好的人，是内在最稳定的人。**

### 说剑——一套话术三种说法

赵文王爱剑，门客三千多人日夜击剑。庄子来见赵王，赵王以为他要论剑。

庄子说："臣有三剑：天子剑、诸侯剑、庶人剑。"

- 天子剑：以燕溪石城为锋，以齐岱为锷——说的是天下格局
- 诸侯剑：以知勇士为锋，以良相为锷——说的是治国用人
- 庶人剑：蓬头突鬓，曼胡之缨——就是赵王现在看的那种斗鸡式比武

庄子用赵王能接受的入口（论剑）引入了他想传达的思想（治国）。

**核心洞察：影响人不是用你的语言说你的话，是用对方的语言说你的话。**

## 情境触发

当你遇到以下状态时，使用此技能：

- 你说的话有道理但没人听
- 你没有职位/资历/资源来"命令"别人
- 想影响上级的决策但只是一个普通员工
- 团队里你不是 leader 但需要推动事情
- 你的"不一样"（背景、身份、风格）被视为劣势

## 行动步骤

### 第一步：停止"推销"——王骀的"立不教"

大多数人试图影响别人的方式是"推"：说更多、论据更强、表达更有力。

**王骀的做法是相反的：他什么都不说，人们自己来。**

这不是教你闭嘴。是教你：**影响力的根基不是你的输出，是你的存在状态。**

检查你自己：
- 你说话时是在分享还是在说服？——说服的姿态本身就在制造对抗
- 你给建议时是"你应该"还是"我的经验是"？——前者是推，后者是放
- 对方不听你的时候你会加大力度吗？——越推越远

### 第二步：修炼"德不形"——哀骀它的内在稳定

**影响力 = 内在稳定性 × 一致性 × 时间**

- **稳定性**：遇到好事不得意忘形，遇到坏事不崩溃——人们会自然靠近一个"稳"的人
- **一致性**：你说的和做的一致，你对上和对下一致，你顺境和逆境时一致
- **时间**：信任需要时间。不要想着一次对话就改变别人

**具体修炼：**
- 有人质疑你时——不急于辩护，先听完，确认理解，再回应
- 出了问题时——不推卸，先说"我来看看怎么解决"
- 有功劳时——不急于邀功，让事实说话
- 每一次这样的表现都在你的"信任账户"里存款

### 第三步：用对方的语言——说剑术

当你确实需要主动影响某个人时，用"说剑"的方法：

**操作：**
1. **找到对方的"剑"**——他关心什么？用什么框架思考？在乎什么？
2. **用他的入口进入**——不要上来就说你想说的，先进入他的世界
3. **在他的框架里展开你的观点**——让他觉得这是他的逻辑的自然延伸

**举例：**
- 想说服一个重视数据的老板 → 不要说"我觉得应该这样做"，而是"从数据看，X指标这样变化，如果我们做Y，预期Z"
- 想影响一个重视关系的合作方 → 不要发一封冷冰冰的分析报告，先建立信任再谈事情
- 想推动一个保守的团队尝试新技术 → 不要讲技术有多先进，讲"这能解决你们现在头疼的X问题"

**三把剑的对应：**
- 天子剑 = 战略层面的人→用格局和趋势说话
- 诸侯剑 = 管理层面的人→用团队和效能说话
- 庶人剑 = 执行层面的人→用具体问题和解决方案说话

### 第四步：把"弱势"变成独特的影响力

哀骀它丑、王骀残——他们的"缺陷"反而让人们看到了更深层的东西。

**操作：**
- 你不是技术背景但想影响技术决策 → 你的"外行视角"可能是团队最缺的（"用户不会这样理解这个功能"）
- 你资历浅 → "新人眼光"能看到老人看不到的惯性问题
- 你不善言辞 → 少说话的人说出来的话权重更高
- 你是少数派 → 独特视角本身就是价值

**关键：不要试图弥补"弱势"变得和主流一样，而是找到"弱势"独有的影响力角度。**

### 第五步：接受影响力的边界

- 你能影响的只有愿意被影响的人
- 有些人不听不是因为你不够好，是因为他不在那个位置
- 最好的影响往往不是当下生效的——种子可能很久以后才发芽
- "虚而往，实而归"——你提供的是空间，不是答案

## 示例

**场景**：你是一个初级工程师，发现团队的部署流程有严重问题，但你说了几次没人当回事。

**影响术做法**：
1. 停止推销：不要再在群里发长消息论述为什么要改——你越说得多，大家越当噪音
2. 修炼稳定性：先把自己的活干到无可挑剔。在每次部署出问题时冷静帮忙解决，不说"我早说过了"
3. 说剑术：找到 tech lead 最关心的事（可能是"减少oncall 报警"）。不说"部署流程有问题"，而说"我注意到最近 3 次 oncall 报警都和部署有关，我整理了一些数据，你有空看看吗？"
4. 用你的位置：初级工程师的视角恰好是"新鲜的眼睛"——"我刚来可能理解不对，但这个流程让我困惑的地方是……"
5. 接受边界：可能 tech lead 看了你的数据还是不动。那就记下来，继续积累信任。也许三个月后大事故发生时，他会想起你说过的话

## 反模式

| 错误 | 为什么错 |
|------|----------|
| 没有影响力所以要"搞政治" | 政治操纵不是影响术，是权术。短期有效长期毁信任 |
| 模仿别人的影响方式 | 王骀不说话但你可能需要说话。找到你自己的方式 |
| 急于见效 | 哀骀它不是一天建立影响力的。急于见效=推销 |
| 以为"做好自己就够了" | 王骀做好自己是因为他内在真的完整了。你如果只是"不出手"但内心焦虑，别人感受得到 |
| 对所有人用同一种方式 | 庄子有三把剑，你至少也要有两把 |

## 退出条件

- 对方不是"没被影响"而是"利益不同意" → 这是谈判问题，用利益交换而非影响力
- 你需要的是权力/权限而不是影响力 → 直接争取正式授权
- 你在用"影响术"回避直接冲突 → 有时候直接说"我反对"比迂回更有效，切换到**心斋术**做正面沟通
