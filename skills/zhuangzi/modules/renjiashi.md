# 处世术 — 在危险复杂的人际环境中生存

> 《人间世》——人活在人间，如何行事而不被毁掉。

本技能包含五个子策略，详见 `skills/renjiashi/` 目录：

| 子策略 | 原典 | 核心 |
|--------|------|------|
| [心斋术](../../renjiashi/modules/xinzhai.md) | 颜回见孔子 | 劝说强势对象前，先清空自己 |
| [不可能任务](../../renjiashi/modules/impossible-mission.md) | 叶公子高使齐 | 进退两难时，只做能做的部分 |
| [伴虎术](../../renjiashi/modules/tiger-companion.md) | 颜阖为太子师 | 与危险人物共事：外在顺从，内心独立 |
| [散木策略](../../renjiashi/modules/scattered-wood.md) | 栎社树 | 降低可见度，以"无用"换生存 |
| [以缺为盾](../../renjiashi/modules/imperfection-shield.md) | 支离疏 | 策略性保留缺陷，免于被征用 |

## 情境触发

当你遇到以下状态时，进入处世术：

- 需要劝说一个比你强势的人
- 被分配了做好没功劳做坏要背锅的任务
- 必须与一个危险/不可预测的人长期共事
- 在高度竞争或政治化的环境中需要自保
- 越能干越被压榨，完美反而成为负担

## 路由

描述你的处境后自动匹配最合适的子策略。也可直接指定：
- `/zhuangzi 处世 心斋` — 劝说场景
- `/zhuangzi 处世 使命` — 两难任务
- `/zhuangzi 处世 伴虎` — 危险上级
- `/zhuangzi 处世 散木` — 自保需求
- `/zhuangzi 处世 支离` — 以缺为盾

## 核心原则

人间世五策的底层逻辑：
- **不是教你逃避，是在不可控中找到可控**
- **保全自我是前提不是目的**
- **顺势不等于无原则——形就是策略，心和是底线**
- **每个策略都有边界，关键是知道何时切换**
