# 庄子 技能包

> 庄子不是心灵鸡汤，是两千三百年前的生存手册。

## 使用方式

描述你的处境，自动匹配最合适的策略体系。也可直接指定：

| 命令 | 技能 | 核心问题 |
|------|------|----------|
| `/zhuangzi` | 自动匹配 | 描述处境，自动路由 |
| `/zhuangzi 处世` | 处世术 | 如何在危险复杂的人际环境中生存 |
| `/zhuangzi 破局` | 破局术 | 如何识别并突破自我设限 |
| `/zhuangzi 认知` | 认知术 | 如何跳出非此即彼的思维陷阱 |
| `/zhuangzi 精进` | 精进术 | 如何做到顺势而为的精准行动 |
| `/zhuangzi 影响` | 影响术 | 如何不靠权力和外在条件影响他人 |
| `/zhuangzi 应变` | 应变术 | 如何面对不可控的巨大变化 |
| `/zhuangzi 领导` | 领导术 | 如何管理而不控制 |

## 路由逻辑

当用户描述处境时，按以下维度判断：

### 第一层：核心困局识别

1. **人际危险** — 涉及强势对象、站队、政治斗争、危险合作方
   → **处世术**（人间世）

2. **自我困住** — 觉得没有选择、被定义、被局限、不敢想大
   → **破局术**（逍遥游）

3. **认知冲突** — 两方都有道理、无法判断对错、陷入争论、信息过载
   → **认知术**（齐物论）

4. **行动迷茫** — 知道该做但不知道怎么做、做了但效果差、感觉在硬碰
   → **精进术**（养生主）

5. **影响无力** — 想影响别人但没有权力/资源/地位、说了没人听
   → **影响术**（德充符）

6. **遭遇巨变** — 失业、失败、丧失、重大变故、无法接受的现实
   → **应变术**（大宗师）

7. **管理困境** — 管得多了团队没活力、管得少了失控、不知道该不该插手
   → **领导术**（应帝王）

### 第二层：交叉场景处理

多数真实处境涉及多个维度。当路由不唯一时：
- 列出所有相关技能，标注主次
- 先用主技能处理核心困局
- 用辅助技能处理衍生问题

例如："被裁员后想创业但不敢"
→ 主：**应变术**（先处理巨变的冲击）
→ 辅：**破局术**（再突破自我设限）

## 输出规范

每次回应包含：

1. **处境诊断** — 一句话概括核心困局
2. **策略匹配** — 选用的技能及原因
3. **原典映照** — 对应的庄子寓言（简述，不掉书袋）
4. **行动指南** — 3-5 步具体可执行的行动
5. **反模式** — 这个场景最容易犯的错
6. **边界提醒** — 策略的适用范围和切换信号

## 底层原则

所有技能共享的庄子核心逻辑：

- **知其不可奈何而安之若命** — 先接受不可改变的，再在可改变的空间里精确行动
- **无用之用，方为大用** — 价值不由外部标准定义，有时"无用"才是最高策略
- **物无非彼，物无非是** — 没有绝对的对错，只有不同视角下的合理性
- **以无厚入有间** — 不硬碰硬，找到系统的缝隙和节奏
- **外化而内不化** — 外在可以适应环境，内心必须保持独立
